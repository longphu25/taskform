# TaskForm — Pages

## index.html (Landing)

- Product intro
- Walrus / Seal / Sui value proposition
- Launch App CTA

## dashboard.html (Admin)

- Wallet connect
- My Forms list
- Submission inbox
- Status and priority management
- Storage Health indicators
- Export JSON/CSV
- Decrypt sensitive fields

## create-form.html (Builder)

- Title and description
- Add/remove fields
- Required toggle
- Sensitive toggle
- Storage duration selector
- Sponsored submission toggle
- Publish form action

## form.html (Public Submit)

- Must load fastest (ultra-light)
- Load schema from Walrus
- Render dynamic fields
- Validate with Zod
- Encrypt sensitive fields only when needed
- Upload attachments only on submit
- Submit to Walrus
- Show success confirmation

## Supported Field Types

- Short text
- Rich text (textarea)
- Dropdown
- Checkbox
- Star rating
- Screenshot upload
- Video upload
- URL
- Confirmation checkbox
