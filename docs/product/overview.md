# TaskForm — Product Overview

## Vision

TaskForm is a Walrus-native feedback operating system for decentralized teams.

## Positioning

> TaskForm is a Walrus-native feedback operating system where decentralized teams can collect
> private feedback, sponsor user submissions, and manage storage lifecycle from form creation
> to action.

## Core User Flow

```text
index.html → dashboard.html → create-form.html → form.html?formId=... → dashboard.html?formId=...
```

## Target Users

**Primary:** Hackathon organizers, Walrus Sessions organizers, Web3 protocol teams, DAO
communities, open-source projects, product teams.

**Secondary:** Bug bounty teams, grant/application teams, internal anonymous feedback teams,
security reporting teams.

## Core Differentiators

1. Walrus-native storage (epoch-based, decentralized)
2. Seal-powered field-level privacy
3. Sponsored submissions (Web2 UX, Web3 ownership)
4. Storage lifecycle management
5. Private admin dashboard
6. CDN-first lightweight Walrus Site

## MVP Success Criteria

- Creator can create and publish form
- Submitter can submit feedback via public link
- Schema stored on Walrus
- Submission stored on Walrus
- Sensitive field encryption demonstrated
- Creator can triage submissions (status/priority)
- Storage expiry visible
- Sponsored submission story demo-ready
- form.html loads quickly
